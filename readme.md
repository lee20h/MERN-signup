## 🙉 설명

MERN(MongoDB, Express, React, Node) 스택을 사용한 웹 사이트에서의 회원가입과 로그인, 로그아웃 상용구 코드

## 👨‍💻 사용법

1. dev.js를 config 폴더 안에 생성
2. mongoDB 정보 dev.js 작성
3. client / server `npm install`로 종속성 다운로드
4. `$ cd server && npm run dev`

## 🏃 스택
- React & Redux
- Node & Express
